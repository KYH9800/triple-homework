# triple
프로젝트 실행을 위해 아래의 터미널에서 아래의 명령어를 순서대로 실행해 주세요.

## git clone
```
git clone https://github.com/KYH9800/triple-homework.git
```

## move directory
```
cd front/
```

## Installation
```
npm install
```

## Start Project
```
npm run dev
```
